# vuejs-h5-wx

> vuejs 开发一套代码,可以构建 H5,小程序(微信,百度,支付宝,今日头条)多个版本

## 具体操作

```bash

# 安装依赖
yarn

# dev 对应平台 h5 微信、百度、头条、支付宝
yarn run dev:h5
yarn run dev:wx
yarn run dev:swan
yarn run dev:tt
yarn run dev:my

# build 对应平台 h5 微信、百度、头条、支付宝

yarn run build:h5
yarn run build:wx
yarn run build:swan
yarn run build:tt
yarn run build:my

```
